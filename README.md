# Payment Middleware

A Payment Middleware

### Technologies

- React
- React-Query
- Typescript
- Tailwind CSS
- .Net 5
- Docker
- Gitlab CI

### Dependencies

- NodeJS
- .Net 5 SDK

#### Commands

- Start frontend: `cd Payment.Client && npm run dev`
- Start backend: `cd Payment.Api && dotnet run`
- Test backend: `cd Payment.Api && dotnet test`
- FullStack: `docker-compose up -d`
